# CyberAgent

MVP автономного Telegram-бота для киберспортивного канала. Бот раз в день собирает свежие новости, отсекает повторы, пишет уникальный пост через OpenAI, генерирует изображение 16:9 с обязательной подписью и публикует результат в Telegram.

## Что уже есть

- ежедневный запуск по `TIMEZONE` и `POST_TIME`
- сбор новостей через Google News RSS с фокусом на источники из `AgentMG.md`
- фильтрация дублей по URL, заголовкам и похожим темам среди последних 30 публикаций
- генерация текста через OpenAI Responses API
- генерация изображения через OpenAI Image API
- обязательная подпись на картинке: `Самара / 5-я Просека, 95`
- публикация в Telegram-канал через Bot API
- SQLite-хранилище истории и логов
- fallback-пост, если свежая уникальная новость не найдена

## Запуск

1. Создайте виртуальное окружение:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

2. Установите зависимости:

```powershell
pip install -r requirements.txt
```

3. Проверьте локальную конфигурацию в `C:\Users\user\Documents\CyberAgent\.env.local`.

4. Запустите один цикл вручную:

```powershell
python main.py --once
```

5. Для постоянной работы оставьте процесс запущенным:

```powershell
python main.py
```

## Важные замечания

- Бот Telegram должен быть администратором канала `@AgroBroker63`, иначе публикация не пройдет.
- Сейчас в качестве агрегатора используется Google News RSS, но источники для поиска ограничены доменами из ТЗ. Это хороший MVP-слой, который можно позже заменить на более точечные парсеры.
- Если генерация изображения упадет, бот все равно отправит текстовый пост.
- История публикаций и логи создаются локально в папках `data/` и `logs/`.
